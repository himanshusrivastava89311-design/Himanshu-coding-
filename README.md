# Himanshu-coding-
🚀Description
This is a simple Python program that swaps two numbers entered by the user without using a third variable.
The program takes two inputs and exchanges their values using Python's multiple assignment feature.
🚀 Features
Takes user input
Swaps two numbers instantly
No extra variable required
Beginner-friendly logic
How It Works
The swapping is done using this line:
a,b= b,a
Purpose
This project is useful for beginners to understand:
Variable assignment
Input/output in Python
Basic logic building
